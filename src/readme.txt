Project Description:

The goal of this project is to design an application that queries a spatial database. 
This project implements spatial data types using Oracle10g, Oracle Spatial features, and Java (JDBC).

Project consists of two Java programs to 1) store (populate.java) and 2) query spatial database (hw2UI.java)

Note: 
1.) hw2UI is main file which is run first and which takes command arguments. Therefore to start the program and compile it use the command:
java hw2UI building.xy people.xy ap.xy

2.) Create Database using createdb.sql to create index,metadata and all tables.
